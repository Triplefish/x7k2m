# ⚠️ 重要说明

## 关于数据源和合规性

### 监管背景

2021年，中国证监会明确要求：
> 基金销售平台**不得向投资者提供盘中估值功能**

目的是避免诱导投资者频繁交易，减少短期炒作行为。

### 本工具的数据源

1. **主数据源**：天天基金网估值API
   - 状态：目前仍可访问
   - 风险：⚠️ **随时可能被关闭**
   - 更新：数据更新至收盘（15:00），非真正"盘中实时"

2. **备用方案**：ETF价格自行计算
   - 适用于ETF联接基金
   - 准确度：中等

3. **兜底方案**：仅显示昨日净值

### 使用建议

#### ✅ 可以使用的场景

- **个人自用**：仅自己查看，不对外分享
- **学习研究**：技术学习和数据分析
- **辅助决策**：作为投资参考，理性使用

#### ❌ 不建议的场景

- **公开传播**：分享给大量用户
- **商业运营**：收费服务或商业产品
- **诱导交易**：用于频繁交易决策
- **第三方服务**：为他人提供估值查询服务

### 合规建议

1. **仅个人使用**
   - 不要公开分享你的维格表链接
   - 不要做成公开网站或小程序
   - GitHub 仓库可设为私有

2. **理性投资**
   - 估值仅供参考，不作为交易依据
   - 避免频繁查看和交易
   - 长期投资，理性决策

3. **数据来源声明**
   - 数据来自公开API，可能随时失效
   - 不保证数据准确性
   - 使用者自行承担风险

### 如果API失效怎么办？

本工具已内置多层备用方案：

1. 天天基金网失效 → 使用ETF价格计算（ETF联接基金）
2. ETF价格失效 → 显示昨日净值
3. 完全失效 → 工具停止使用

### 建议的替代方案

如果天天基金网API被完全关闭，建议：

1. **使用基金公司官方App**
   - 各基金公司的官方渠道
   - 合规且稳定

2. **支付宝/微信理财**
   - 大平台数据更新及时
   - 不提供盘中估值，符合监管

3. **只关注每日净值**
   - 收盘后查看当日净值
   - 更适合长期投资理念
